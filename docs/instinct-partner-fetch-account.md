# Instinct Partner API Reference: Fetch Account

Saved from:

- `https://docs.instinctvet.com/reference/instinct-partner-fetch-account`

Retrieved:

- `2026-04-10`

Endpoint documented on the official page:

- `GET https://partner.instinctvet.com/v1/accounts/{id}`

Notes from the reference page:

- Page title: `Fetch Account`
- Description: fetches an account by ID
- Path parameter:
  `id` is a required UUID account ID
- Auth shown on the page:
  `Bearer`
- Documented responses shown on the page:
  `200` for account response and `404` for account not found

This file is a project-local note, not a full mirror of the vendor documentation.
